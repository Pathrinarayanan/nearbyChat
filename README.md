# Nearby Chat

Personal Kotlin Android experiment around nearby / local chat UI and messaging flows.

Learning project — not a published product.

## Stack

Kotlin · Android

## Run locally

```bash
git clone https://github.com/Pathrinarayanan/nearbyChat.git
cd nearbyChat
```

Open in Android Studio, sync, run.
